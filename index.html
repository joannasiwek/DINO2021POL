Hello World

git clone /path/to/repository - klonuje repozytorium z podanej ścieżki do obecnego folderu na dysku

git init - Polecenie tworzy nowe repozytorium git lub resetuje aktualne

git config --global user.email "" - Komenda służy do konfiguracji adresu email. Opcja --global zmienia adres email dla wszystkich projektów git.

git commit "filename" -> zapisuje migawkę plików naszego projektu. Commit zawiera wszystkie pliki projektu. W przypadku plików, które się nie zmieniły, git przechowuje tylko łącze do poprzedniego identycznego pliku, który jest już zapisany.

git checkout -b "branch" - tworzy nową gałąź o nazwie "branch" i przełączą się na nią

git branch -d "branch" - usuwa branch o podanej nazwie w tym przypadku: "branch"

git config --global color.ui true -> Komenda ustawiona na wartość true koloruje informacje zwrotne przeznaczone dla użytkownika w terminalu.

git fetch - pobieranie zmian z odpowiednich gałęzi zdalnych do gałęzi lokalnych je śledzących bez scalania zmian

git reset HEAD - polecenie, które przywraca stan repozytorium do zatwierdzenia HEAD.

git diff --staged - pokazuje zawartość poczekalni która trafi do repozytorium wraz z najbliższym zatwierdzeniem. Porównuje zmiany poczekalni z ostatnią zmianą

git push - komenda służy do pushowania zmian z lokalnego repozytorium do repozytorium zdalnego

'git log --graph --all' - komenda pokazuje logi commitów, opcja --graph rysuje graf historii commitów

git init - polecenie tworzy puste repozytorium GIT lub reinicjalizuje już istniejące
